# AC Telemetry

Crude Assetto Corsa Telemetry logger

# get started

    python3 logger.py [IP address]

Dumps telemetry per lap into the `out` directory


# plotting

    python3 -m venv py
    py/bin/pip install bokeh numpy
    py/bin/python plot-simple.py
